# dotnet-arch-analyzer

AI-powered architecture and code analysis tool for modern .NET projects.

`dotnet-arch-analyzer` analyzes C# codebases and helps developers understand architecture, dependencies, and potential design issues in modern .NET applications.

## Features

- Analyze C# project structure
- Detect architectural issues
- Generate architecture reports
- Visualize dependency graphs

## Usage
```bash
dotnet-arch analyze <projectPath>
dotnet-arch report <projectPath>
dotnet-arch graph <projectPath>
```
## Example:
```bash
dotnet-arch analyze .
```
